callAction ( $  action,
$  args = array(),
$  redirect = true 
)

Calls an action, by default with a 303 redirect - implementing the POST-REDIRECT-GET pattern. The redirect preserves only messages.

Parameters:
$actionthe action's name, e.g.: 'homepage'.
$argsarray of arguments past to the action function.
$redirect(optional) Defaults to true. If false, the action will be called directly.
